# Font Fantasy
Font Fantasy is a tall and slender 16pt pixel art font with simple curves giving it a subtle fantasy feel.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 16px
- Scales: 16, 32, 48, 64, etc
- Ascender Max: 6px
- Descender Max: 3px
